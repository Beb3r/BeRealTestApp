# BeRealTestApp

The app has 4 screens: login, browser, settings and visualizer for image.
Fully modularized, made on kotlin 1.8, clean arch and mvvm.
Stack:
- Compose,
- Coroutines/Flow,
- Hilt,
- Room,
- Accompanist,
- OkHttp/Retrofit,
- KotlinX Serialization
- Coil,
- Timber
